# 🍽️ Sistema de Gestión de Restaurante

Aplicación web desarrollada para la administración de operaciones en un restaurante, permitiendo gestionar pedidos, menú, mesas y facturación de manera eficiente mediante una interfaz moderna e interactiva.


## 🚀 Funcionalidades

### 🧾 Gestión de Pedidos
- Creación de pedidos por mesa
- Selección de platos desde el menú
- Cálculo dinámico del total
- Finalización de pedidos

### 🍔 Gestión de Menú
- Registro de platos
- Asignación de precios
- Clasificación por categorías
- Visualización del menú disponible

### 🪑 Gestión de Mesas
- Registro de mesas
- Control de estado (ocupada / libre)
- Asociación de pedidos a mesas

### 💰 Facturación
- Cálculo automático de totales
- Generación de pedidos finalizados

### 📦 (Inventario)
- Control básico de inventario


## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS
- JavaScript


## 💡 Características

- Interfaz moderna y amigable
- Manejo dinámico del DOM
- Validaciones de entrada
- Lógica de negocio aplicada
- Estructura modular del sistema


## 🎯 Objetivo del Proyecto

Desarrollar una aplicación web que simule el funcionamiento de un sistema real de restaurante, aplicando conceptos de desarrollo frontend, lógica de programación y gestión de datos.


## 📌 Posibles Mejoras

- Integración con base de datos real
- Sistema de autenticación de usuarios
- Generación de reportes
- Implementación de backend (API REST)
- Control avanzado de inventario
