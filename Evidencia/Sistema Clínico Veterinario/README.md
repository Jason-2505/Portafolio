# 🐾 Sistema Clínico Veterinario

Aplicación desarrollada en Java para la gestión de una clínica veterinaria, permitiendo administrar mascotas, propietarios, citas y consultas médicas.

## 🚀 Funcionalidades
- Registro de clientes y mascotas
- Gestión de citas veterinarias
- Historial de consultas
- Control de tratamientos

## 🛠️ Tecnologías
- Java
- Eclipse IDE
- (SQL si usaste base de datos)
