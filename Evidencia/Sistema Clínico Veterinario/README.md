# 🐾 Sistema Clínico Veterinario

Aplicación desarrollada en Java para la gestión integral de una clínica veterinaria, permitiendo administrar información de clientes, mascotas, citas y consultas médicas, simulando el funcionamiento de un sistema real en el área de salud animal.

## 🚀 Descripción

Este proyecto consiste en el desarrollo de un sistema clínico veterinario utilizando programación orientada a objetos en Java. El sistema permite gestionar de manera eficiente los procesos principales de una clínica, incluyendo el registro de pacientes (mascotas), control de propietarios, programación de citas y seguimiento de consultas médicas.

El objetivo es representar un entorno real donde se aplican conceptos de desarrollo de software, organización de datos y lógica de negocio.

## 🧩 Funcionalidades

- 👤 Registro de clientes (propietarios)
- 🐶 Gestión de mascotas (pacientes)
- 📅 Programación de citas veterinarias
- 🩺 Registro de consultas médicas
- 💊 Gestión de tratamientos
- 📋 Historial clínico de cada mascota
- 🔍 Búsqueda de información de clientes y mascotas


## 🛠️ Tecnologías utilizadas

- Java
- Programación Orientada a Objetos (POO)


## 🧱 Estructura del sistema

El sistema está organizado en clases que representan las entidades principales:

- `Cliente`
- `Mascota`
- `Cita`
- `Consulta`
- `Tratamiento`

Estas clases permiten modelar el comportamiento del sistema y gestionar la información de manera estructurada.


## ⚙️ Características técnicas

- Uso de clases y objetos para representar entidades del sistema
- Encapsulamiento de datos mediante atributos y métodos
- Uso de estructuras dinámicas (listas) para almacenamiento de información
- Aplicación de lógica de negocio para gestión de citas y consultas


## 🎯 Objetivo del proyecto

Desarrollar un sistema funcional que simule la gestión de una clínica veterinaria, aplicando conceptos de programación orientada a objetos y buenas prácticas en el desarrollo de software.


## 🚀 Posibles mejoras

- Integración con base de datos (SQL Server u Oracle)
- Implementación de interfaz gráfica (Java Swing o JavaFX)
